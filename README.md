## Guiding Principles

* **Simplicity**
  > Complexity transforms readable programs into unreadable ones. Complexity has ended many software projects.

* **Readability**
  > Code is written for humans to read; reading time far exceeds writing time. Programs must be maintainable, and readability is the first step.

* **Productivity**
  > With numerous tools and standard libraries, most tasks can be completed simply and conveniently.
  > Compilation is fast enough to provide the efficiency of dynamic languages without facing their reliability issues.
  > Built-in coding standards ensure consistent team code and help developers discover and avoid potential errors.
